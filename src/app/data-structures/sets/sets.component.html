HULGAD

Mida selles peatükis õpime?
1. Mis on hulgad ja milleks neid kasutatakse?
2. Millised on ainult hulkadele iseloomulikud tehted?
3. Milliseid funktsioone ja meetode on hulkadel?
4. Kuidas hulka tsüklis läbida?
Järgmiseks andmestruktuuriks, millega tutvume, on hulk. Hulk hoiab endas üksikuid elemente, kuid erinevalt järjendist ei ole elemendid siin kindlas järjekorras. Samuti pole hulgas korduvaid elemente. Hulkadega käivad käsikäes neile iseloomulikud hulgatehted, millega samuti tutvume.
Inglise keeles on hulk set ja just sama nimega on ka vastab Pythoni tüüp. Hulga ümber on sarnaselt sõnastikule loogelised sulud. Kuna tühja sõnastiku loomiseks juba kasutatakse tühje loogelisi sulge, siis hulka niimoodi luua ei saa. Saame kasutada vaid hulga loomise meetodit set(). Võime hulka lisada sama elementi mitu korda, kuid kui proovime näiteks hulka väljastada, siis näeme, et seal on hoitakse iga elementi ikkagi ühekordselt.
arvud = {{'{'}}1, 5, 3, 1{{'}'}} # kuigi luues paneme sõnastikku kaks number 1, siis jääb sinna neist ainult üks
 värvid = {{'{'}}"sinine", "punane", "kollane"{{'}'}}
 segamini = {{'{'}}1, "esmaspäev", 14.5, True{{'}'}}
 tühi = set() # tühi = {{'{}'}} loob tühja sõnastiku
Võid proovida käivitada järgmist koodijuppi. Esiteks märkad, et korduvad elemendid on tõesti eemaldatud ja teiseks, et elementide järjekord ei pruugi olla sama, mis hulka luues.
print({{'{'}}4, 7, 7, 5, 9, 2, 3, 2, 7{{'}'}})
{{'{'}}2, 3, 4, 5, 7, 9{{'}'}}

Hulkadega kasutatavad operatsioonid, funktsioonid ja meetodid

Operatsioonid
Hulkadel on omajagu operatsioone, mis on ainult neile omased ja tänu millele on hulgad teatud olukordades väga kasulikud. Meil on kolm hulka A = {{'{'}}1, 3, 5, 2, 6, 7{{'}'}}, B = {{'{'}}1, 2, 8, 9, 4{{'}'}} ja C = {{'{'}}2, 6{{'}'}}, mis on salvestatud vastavalt muutujatesse a, b ja c.
Avaldis Väärtus Selgitus Joonis
2 in a True Kontrollib, kas element on hulgas
3 not in b True Kontrollib, ega elementi ei ole hulgas
a == {{'{'}}3, 6, 7, 1, 2{{'}'}} True Kontrollib, kas hulgad on võrdsed. Kusjuures elementide järjekord pole oluline
a != b True Kontrollib, ega hulgad ei ole võrdsed. Elementide järjekord pole oluline.
a & b {{'{'}}1, 2{{'}'}} Ühisosa ehk hulk elementidest, mida sisaldavad mõlemad hulgad.
a | b {{'{'}}1, 3, 5, 2, 6, 7, 8, 9, 4{{'}'}} Ühend ehk hulk elementidest, mida sisaldab vähemalt üks hulkadest. Ehk siis elemendid mis on hulgas A, hulgas B või neis mõlemas.
a - b {{'{'}}3, 5, 6, 7{{'}'}} Vahe ehk hulk elementidest, mis on hulgas A, aga mida ei ole hulgas B. Teisisõnu hulk A, millest on ära võetud A ja B ühisosa.
a ^ b {{'{'}}3, 5, 6, 7{{'}'}} Sümmeetriline vahe ehk hulk elementidest, mis on hulgas A või hulgas B, aga mitte neis mõlemas. Teisisõnu ühend, millest on ära võetud ühisosa.
c < a True Kontrollib, kas C on hulga A range alamhulk ehk kas hulk C koosneb ainult elementidest, mis on ka hulgas A. Kusjuures C ja A ei tohi olla võrdsed.
a > c True Kontrollib, kas A on hulga C range ülemhulk ehk kas kõik hulga C elemendid on olemas ka hulgas A. Kusjuures C ja A ei tohi olla võrdsed.
c <= a True Kontrollib, kas C on hulga A mitterange alamhulk ehk kas hulk C koosneb ainult elementidest, mis on ka hulgas A. Hulgad tohivad olla võrdsed.
a >= c True Kontrollib, kas A on hulga C mitterange ülemhulk ehk kas kõik hulga C elemendid on olemas ka hulgas A. Hulgad tohivad olla võrdsed.

Hulk ja tsükkel
Kuigi me ei saa hulgast indeksitega elemente kätte, siis lihtsalt for-tsükliga elemente läbi käia saame ikka.

 for päev in nädalapäevad:

Seljakottide võrdlus
Meeli ja Mihkel kõnnivad koolist koju ja jäävad arutama, mis neil kummalgi seljakotis on. Neid huvitab, et millised esemed neil kahepeale kokku on, millised asjad on neil mõlemal olemas ning mis asjad on näiteks ainult Meelil või ainult Mihklil. Mõlema seljakottide sisud on kirja pandud hulkadena. Leia ja väljasta, mis asjad neil kahepeale kokku on, millised asjad on mõlema koolikotis, mis asju leidub Meeli koolikotis, mida Mihklil ei ole, ja vastupidi.
